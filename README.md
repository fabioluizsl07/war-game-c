# War Game - Simulação com Missões Estratégicas

## Funcionalidades Implementadas
- Alocação dinâmica de territórios
- Sistema de jogadores com cores
- Missões estratégicas sorteadas
- Turnos alternados
- Validação de ataques
- Verificação de vitória
- Gerenciamento completo de memória

## Como Compilar
```bash
gcc -o war main.c -Wall -Wextra -std=c11
./war
